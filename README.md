# README

Provide a script file to configure proxy settings.

# Install

```sh
git clone https://github.com/Lockdef/ProxySettings.git
mv ./ProxySettings/ ~/sh/
```
### Config

Edit .ProxyConfig
```
PROXY="<URL>:<PORT>"
```

Example
```
PROXY="hogehoge.com:8888"
```

### Use
```
sh ./<File Name>.sh
```

Example
```
sh ./BashSetProxy.sh
```
